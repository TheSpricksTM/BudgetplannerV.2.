# BudgetPlanner — Aiman Mahgoub

Een gratis, interactieve budgetplanner voor mensen die financiële stabiliteit zoeken. Geen account, geen abonnement, geen datadeling.

🔗 **Live:** [aiman-mahgoub.github.io/budgetplanner](https://aiman-mahgoub.github.io/budgetplanner)

---

## Pagina's

- **Dashboard** — Overzicht, KPI's en grafieken
- **Budgetinvoer** — Maandelijkse inkomsten en uitgaven bijhouden
- **Grafieken** — Diepgaande analyses en trends
- **Maandoverzicht** — Per maand inzoomen
- **Spaardoelen** — Doelen instellen en voortgang bijhouden
- **Prognose** — Financiële simulatie tot 20 jaar vooruit
- **Instellingen** — Dark mode en kleurthema's

## Gebruik

Open `index.html` voor de frontpage of `planner.html` voor de budgetplanner direct.

Alle data wordt opgeslagen in de browser (localStorage) — volledig privé.
